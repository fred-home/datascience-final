Universal Studios Review Sentiment Analysis

Overview

I am using Natural Language Processing to analyze reviews from Universal Studio theme parks in Florida, Japan, and Singapore. I am using a Kaggle dataset Reviews of Universal Studios for this analysis.

I will use the VADER (Valence Aware Dictionary for Sentiment Reasoning) model from NLTK to determine the sentiment of the review text and compare this with the rating of the review.
